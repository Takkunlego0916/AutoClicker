# AutoClicker

Windows / Android 対応のシンプルなオートクリッカーです。

---

## ダウンロード

実行ファイルは以下から取得してください。

👉 GitHubの「Releases」からダウンロード

* Windows: `AutoClicker.exe`
* Android: `AutoClicker.apk`

---

## 対応環境

* Windows OS 10 / 11
* Android OS（アクセシビリティ機能対応端末）

---

## 主な機能

* 指定位置の自動クリック
* フローティング操作パネル
* クリック間隔の調整
* Androidではアクセシビリティサービス対応

---

## 使い方

### Windows

1. `.exe` を起動
2. クリック位置を設定
3. スタートで自動クリック開始

---

### Android

1. APKをインストール
2. アプリを起動
3. アクセシビリティサービスを有効化
4. フローティングUIから操作

---
